配置环境变量

临时配置环境变量
[root@sou_mysql ~]# export  MYSQL_HOME=/usr/local/mysql825/mysql8
[root@sou_mysql ~]# export  PATH=$PATH:$MYSQL_HOME/bin/

永久环境变量
- 所有用户都可以使用
vim /etc/profile
......
export  MYSQL_HOME=/usr/local/mysql825/mysql8
export  PATH=$PATH:$MYSQL_HOME/bin/

刷新环境变量
[root@sou_mysql ~]# source /etc/profile
- 普通的用户的环境变量
[abbott@y36-server ~]$ vim .bash_profile
